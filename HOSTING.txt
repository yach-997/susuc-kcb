# 一次性配置（只需做一次）：
# 1. 打开 https://my.shipstatic.com → 账户/设置 → 复制 API Key
# 2. 打开 https://github.com/yach-997/susuc-kcb/settings/secrets/actions
# 3. 新建/更新 Secret：名字 SHIP_API_KEY，值粘贴 Key
# 4. 在 ShipStatic 控制台「域名」里创建 susuc-kcb，并关联到最新部署
#    或把 API Key 发给我，我帮你在命令行绑定
# 绑定成功后同学只收藏 https://susuc-kcb.shipstatic.com/ 即可。
