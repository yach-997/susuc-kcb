手机更新不了？按这个做

桌面图标里的旧缓存会劫持所有页面，所以 App 内清理常常无效。

0为什么无痕是新的、普通模式是 1.0.0？

普通模式里装着很早以前的 Service Worker + 缓存，会强行打开旧页面。 无痕模式是干净档案，没有这份旧缓存，所以能进最新版。

要让普通模式也变新：必须清掉本站数据（见下面第 2 步），或先一直用无痕。

1先确认无痕是新版

Chrome 菜单 → 新的无痕式标签页 → 打开：

无痕里点我进入课表

设置 → 关于 应显示 1.3.11 或更高。

2让普通模式也更新（清本站数据）

Chrome：

1）用普通标签页（非无痕）打开上面的网址

2）点地址栏左侧的 锁头 / 天平 / 网站图标

3）进入 网站设置 → 清除并重置 / 删除数据

4）删掉桌面上的旧课表图标

5）再重新打开网址；设置里应变成 1.3.11

若旧地址一直卡住，请改用： https://susuc-kcb.shipstatic.com

华为 / 小米浏览器：设置 → 隐私与安全 → 清除数据 → 勾选「缓存」「网站数据」后清除，再重开。

尝试自动清理页

3清完后

在新版页面里再「添加到主屏幕」。旧图标请删掉，否则点进去还是旧缓存。
